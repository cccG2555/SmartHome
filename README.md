# 라즈베리파이 도어 시스템 - 2025.06.20.

4x4 키패드, PIR 인체감지센서, 서보모터를 이용한 자동 도어 시스템입니다.

## 기능

1. **키패드 인증**: 4x4 매트릭스 키패드에서 올바른 비밀번호(기본: "1234")를 입력하면 문이 열립니다.
2. **자동 문 닫기**: 문이 열린 후 5초가 지나면 자동으로 닫힙니다.
3. **PIR 센서 감지**: 문이 열린 상태에서 PIR 센서가 인체를 감지하면 3초 후 자동으로 문이 닫힙니다.

## 하드웨어 연결

### 키패드 (4x4 매트릭스)
- 행 핀: GPIO 2, 3, 4, 5
- 열 핀: GPIO 6, 7, 8, 9

### PIR 센서
- 신호 핀: GPIO 22
- LED 표시: GPIO 23

### 서보모터
- 제어 핀: GPIO 3

## 컴파일 및 실행

### 모든 프로그램 컴파일
```bash
make all
```

### 메인 프로그램만 컴파일 및 실행
```bash
make run
```

### 개별 모듈 실행
```bash
# 키패드 테스트
./key

# PIR 센서 테스트
./pir

# 서보모터 테스트
./door
```

## 비밀번호 변경

`main.c` 파일에서 다음 부분을 수정하세요:

```c
#define PASSWORD "1234"  // 원하는 비밀번호로 변경
#define PASSWORD_LENGTH 4  // 비밀번호 길이에 맞게 수정
```

## 동작 방식

1. **키패드 스레드**: 지속적으로 키패드 입력을 모니터링합니다.
2. **PIR 스레드**: PIR 센서 상태를 모니터링합니다.
3. **메인 스레드**: 두 스레드를 관리하고 전체 시스템을 제어합니다.

## 주의사항

- 라즈베리파이에서 실행해야 합니다.
- GPIO 라이브러리(libgpiod)가 설치되어 있어야 합니다.
- 적절한 권한으로 실행해야 합니다 (sudo 권장).

## 설치된 라이브러리

```bash
sudo apt-get install libgpiod-dev
```

## 문제 해결

1. **GPIO 권한 오류**: `sudo`로 실행하세요.
2. **컴파일 오류**: libgpiod-dev가 설치되어 있는지 확인하세요.
3. **센서 인식 안됨**: GPIO 핀 연결을 확인하세요.
