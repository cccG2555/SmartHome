# 스마트홈 미니 시뮬레이터 (Smart Home Mini Simulator) <!--[cite: 4] -->


아두이노 Uno와 라즈베리파이 5를 활용하여 출입 제어, 자동 조명, 화재 감지 기능을 구현한 스마트홈 미니 시뮬레이터입니다. <!--[cite: 4] --> 두 보드는 실시간 통신 없이 병렬적으로 독립 실행되며 각자의 지정된 역할을 수행합니다. <!--[cite: 4] -->

<br>

## 📌 주요 기능 <!--[cite: 4] -->

1. **자동 출구 개방**: 실내 측 초음파 센서가 사람의 접근을 감지하면 라즈베리파이가 서보모터를 작동시켜 출구를 자동 개방합니다. <!--[cite: 4] -->
2. **비밀번호 도어락**: 4x4 키패드에 올바른 비밀번호를 입력하면 라즈베리파이가 입력값을 검증하고 서보모터를 작동시켜 문을 엽니다. <!--[cite: 4] -->
3. **자동 조명 제어**: 아두이노에 연결된 조도 센서가 주변 밝기를 측정하여 기준값보다 어두울 경우 LED를 자동 점등합니다. <!--[cite: 4] -->
4. **화재 감지 경보**: 불꽃 감지 센서가 화재를 감지하면 아두이노가 부저와 경보 LED를 통해 즉각적인 알림을 발생시킵니다. <!--[cite: 4] -->

<br>

## 🛠 하드웨어 핀 맵 (Pin Mapping) <!--[cite: 4] -->

시스템은 기능별로 센서와 액추에이터를 분리하여 두 개의 제어 보드에 연결합니다. <!--[cite: 4] -->

### 🍓 라즈베리파이 5 (출입 통제 시스템) <!--[cite: 4] -->
- **GPIO 2 ~ 9**: 4x4 키패드 (행/열 스캔용) <!--[cite: 4] -->
- **GPIO 10**: 서보모터 제어 <!--[cite: 4] -->
- **GPIO 17, 27**: 초음파 센서 (Trigger / Echo 핀) <!--[cite: 4] -->

### ♾️ 아두이노 Uno (환경 모니터링 시스템) <!--[cite: 4] -->
- **A0 (아날로그)**: 불꽃 감지 센서 <!--[cite: 4] -->
- **A1 (아날로그)**: 조도 센서 <!--[cite: 4] -->
- **D2 (디지털)**: 화염 경보 LED <!--[cite: 4] -->
- **D4 (디지털)**: 자동 조명 LED <!--[cite: 4] -->
- **D8 (디지털)**: 능동 부저 <!--[cite: 4] -->

<br>

## 💻 소프트웨어 및 실행 방법 <!--[cite: 4] -->

### 라즈베리파이 (출입 통제) <!--[cite: 4] -->
라즈베리파이 측 제어 프로그램은 C 언어로 작성되어 있습니다. <!--[cite: 4] --> 초음파 센서 거리 측정 및 키패드 입력을 기반으로 서보모터를 제어합니다. <!--[cite: 4] -->

### 아두이노 (환경 감지) <!--[cite: 4] -->
아두이노 프로그램은 라즈베리파이 환경에서 컴파일 후 다운로드하여 실행됩니다. <!--[cite: 4] -->
1. `avr-gcc`를 이용하여 아두이노 프로그램을 컴파일합니다. <!--[cite: 4] -->
2. `avrdude`를 통해 아두이노 보드로 hex 프로그램을 업로드합니다. <!--[cite: 4] -->
3. 아두이노는 `loop()` 함수를 통해 센서 값을 지속적으로 읽어 부저와 LED를 자체 제어합니다. <!--[cite: 4] -->
